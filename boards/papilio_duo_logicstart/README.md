Papilio-DUO
==============
Compact Papilio FPGA board with a Spartan6 Papilio FPGA on the top and an atmega32u4 Arduino derived design on the bottom.

Features
--------
* Arduino Mega Footprint
* Spartan 6 FPGA
* SRAM 512K-2MB
* FT2232H USB
* atmega32u4 AVR

See also: https://github.com/GadgetFactory/Papilio-DUO
